**Non-negotiable (affects SEO, UX, and your credibility as a builder):**

- **Core Web Vitals:**
  - LCP (Largest Contentful Paint): < 2.5s
  - FID (First Input Delay): < 100ms
  - CLS (Cumulative Layout Shift): < 0.1
  - Tools: Vercel Speed Insights, Lighthouse

- **Accessibility (WCAG 2.1 AA minimum):**
  - Semantic HTML (nav, main, article, section)

  - Video captions (your sports videos)

  - Touch targets ≥ 44px
  - Readable on all screen sizes
  - Test on real devices (not just Chrome DevTools)

#### **4.2 Key Implementation Details**

**Real-Time World Map:**
```typescript
// Convex schema
export const schema = {
  recommendations: defineTable({
    location: v.object({ lat: v.number(), lng: v.number() }),
    text: v.string(),
    email: v.optional(v.string()),
    likes: v.number(),
    createdAt: v.number(),
    approved: v.boolean(),
  }).index("by_approval", ["approved"]),
};

// Mutation: Save recommendation
export const saveRecommendation = mutation({
  args: { location: v.object(...), text: v.string(), email: v.optional(v.string()) },
  handler: async (ctx, args) => {
    const id = await ctx.db.insert("recommendations", {
      location: args.location,
      text: args.text,
      email: args.email,
      likes: 0,
      createdAt: Date.now(),
      approved: false, // You moderate
    });
    return id;
  },
});

// Query: Get all approved recommendations
export const getApprovedRecommendations = query({
  handler: async (ctx) => {
    return await ctx.db
      .query("recommendations")
      .filter((q) => q.eq(q.field("approved"), true))
      .collect();
  },
});
```
